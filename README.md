# SCRCambio

Port de mi anterior aplicación sin lanzar: Cambio, hecha con MIT AppInventor y diseñada para Android. Pero que decidí portear a Flutter, así pudiendo crear para otras plataformas.

## ¿Que hace?

La idea principal es una app para usar la pantalla de tu teléfono como luz más tenue, ya que la linterna me parece demasiado fuerte, como para usar mientras se dibuja en una libreta a las 3AM. Poco después debería tener opciones de configuración extra.

También es un reto para aprender a usar Flutter, adaptar el código dependiendo de la plataforma, otras funciones de GitHub como actions y como parte de mi Portafolio personal.
